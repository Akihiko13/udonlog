# 🍜 うどログ 開発記録

香川のさぬきうどんを記録・交流するサービス「うどログ」（udolog.com）の開発ログ。
日付ごとに主な実装をまとめています。

## 2026年

**6月13日**：トップページ（LP）を作成・GitHubで公開
**6月14日**：うどん画像を追加、デザイン全面リニューアル、メール登録オプション追加
**6月15日**：会員登録ページにメール入力欄、お店選択ページを追加
**6月16日**：ログイン／記録投稿／店舗詳細ページを追加、ページ間リンク整備
**6月17日**：マイページにスタンプカード機能、スタンプ帳＋達成バッジを実装
**6月18日**：記録の保存（localStorage）、週/月/年の杯数集計、利用規約・プライバシー、店舗データのCSV管理化
**6月19日**：メニュー選択式、店ごとのおすすめ度、共通ヘッダー、Xシェア機能、五十音順並べ替え・ひらがな検索
**6月20日**：エリア制覇バッジ、閉店フラグ、全店制覇率パネル、スクロール演出
**6月21日**：達成バッジ難易度調整、訪問回数バッジ、常連カード、お知らせ機能、アイコン化で大幅軽量化、ブログリンク・駐車場欄、店舗を108件に拡充
**6月22日**：ブログ記事から営業時間・定休日・駐車場を取り込み、おすすめ度の記録者限定化、定休日表示の整形
**6月23日**：会員機能の土台（DB＋認証API）を構築、サービス名を「うどんログ」→「うどログ」に改名、記録・評価API、ファビコン一式
**6月24日**：会員機能を全ページ連携（バッチ1〜3）＝ログイン・記録・マイページ・店一覧・店舗詳細をサーバー保存に。みんなの投稿（全会員）＋Xリンク交流、活動エリア・Xユーザー名、パスワード強度メーター、.htaccess整備
**6月25日**：運営者情報・お問い合わせ（info@udolog.com）、OGP/シェア画像、記録の削除機能を実装
**6月26日**：会員の公開プロフィール（user.html）＋プロフィール写真、プロフィール編集、退会機能、パスワード再設定（メール式）
**6月27日**：メール認証（6桁コード）、アカウント設定（メール表示／パスワード変更）、推しのうどん屋（最大3軒）、Googleログイン
**6月28日**：記録写真の実保存（サーバーアップロード＋サムネ表示）、スタンプ帳を横スクロール式に刷新（最新カードは展開＋過去カードは矢印/ホイール対応のカルーセル、冊カウント表示）、スタンプ帳の枠・背景を整理、記録削除時に写真ファイルも削除、「記録したお店」をエリア別・「記録の履歴」を月別の折りたたみ表示に、食べたうどんの選択を必須化（フロント＋サーバー）
**6月29日**：店舗詳細のおすすめ度を実データ集計に、常連店をプロフィールに表示、常連マークを王冠アイコンに統一、常連カードのスタンプに記録日を表示、店舗詳細の「代表メニュー」を記録集計の「人気メニュー」に変更、みんなの投稿に「いいね」機能を追加（マイページ等で獲得数も表示）、記録の編集機能を実装、共通ヘッダーの読み込みを改善（表示の遅れ・レイアウトのガクつきを解消）、店舗詳細ページに構造化データ（JSON-LD）を設置、お知らせを拡充（一覧は10件ずつ表示）、トップページの紹介文・デモ表示を実装に合わせて整理、Googleアナリティクス（GA4）を全ページに設置
**6月30日**：イメージカラーのアクセシビリティ改善（ボタン・本文リンクを#8A5510に統一してWCAG AA達成）、達成バッジを4軸・25種に再設計（香川約500軒を前提に広さ/深さ/行動/エリア軸へ拡充）、記録ページの公開設定の絵文字をTablerアイコンに統一（🌐→ti-world / 🔒→ti-lock）、非公開記録はバッジ・制覇率などの公開集計に反映されない旨の注意書きを追加、バッジ絵文字を内容に合うものへ修正＋カテゴリ別セクション表示に変更、空状態4箇所にマスコット「うどまる」（しょんぼり表情のSVG）を追加、コンテンツ画像を images/ フォルダに整理して全ページの参照を更新
**7月1日**：お知らせに6/29〜6/30分の抜け（記録の編集・いいね機能・バッジ再設計・非公開注意書き）を追記、トップページのstats-barを「523店を網羅予定（※少しずつ掲載数を増やしています）」に修正・「食べたうどんを記録」に文言変更、公式Xアカウント（@udolog_com）を開設し全ページのtwitter:siteタグおよびフッターリンクを更新
**7月2日**：管理ダッシュボード（admin.php）を追加＝会員数・記録数・登録推移をパスワード保護で確認（パスワードはconfig.php管理・noindex）、公開範囲「自分のみ」時の集計反映を全APIで検証（マイページは全記録／公開プロフィール・みんなの投稿は公開のみ、で正しく動作を確認）、robots.txt と sitemap.xml を追加（api・uploadsをクロール除外／営業中103店を含む110URLをサイトマップ化）、Google Search Consoleにサイト登録・所有権確認（トップページにmetaタグ設置）、退会時に記録写真が消し残るバグを修正、プロフィールURLをユーザー名方式（udolog.com/＜ユーザー名＞）に変更＝連番ID露出と会員数推測を回避（登録時にユーザー名を設定・Google登録は自動生成＋後から変更可・.htaccessできれいURLをリライト）、リリース前の通しテストを完了
